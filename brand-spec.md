# Brand Spec — Servicios de Luna

## Color Tokens

```css
--bg:      #FFFFFF       (Blanco Pureza — fondos principales)
--surface: #F0F4F8       (Gris Plata — fondos alternos, tarjetas)
--fg:      #0B1A2E       (Azul Noche Profundo — texto principal)
--muted:   #1E3A5F       (Azul Luna — texto secundario, fondos confianza)
--border:  #D4A373       (Dorado Cálido — bordes premium)
--accent:  #2EC4B6       (Turquesa Salud — botones, CTAs, enlaces)
```

## Font Stacks

- **Display:** 'Montserrat', 'Inter', -apple-system, system-ui, sans-serif
- **Body:** 'Inter', 'Open Sans', -apple-system, system-ui, sans-serif
- **Mono:** ui-monospace, 'JetBrains Mono', monospace

## Layout Posture

1. **Radii:** 12–16px consistent border-radius on cards, buttons, inputs
2. **Glassmorphism:** subtle backdrop-blur on overlay cards and nav
3. **Shadow:** soft box-shadows for depth without saturation
4. **Accent budget:** Turquesa used exclusively for actions (buttons, links, CTAs), never for large backgrounds. Dorado for premium highlights only.
5. **Dark sections:** Azul Noche Profundo for authority sections (servicio estrella, cobertura, footer). White text on dark.
6. **Whitespace:** generous padding (section min 80px vertical), 65ch max body width
